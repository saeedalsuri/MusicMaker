# MusicMaher

#### The Default Template for a Neutraliojs app

This is the default template for a Neutraliojs app that could be used for any type of desktop or mobile application.


- Using Neutraliojs SDK
 - Creating windows
 - Mobile apps

- Plugins


### Support

If you have any issues with the template, please log an issue at https://github.com/neutraliojs/maximum/issues.
